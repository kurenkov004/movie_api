For_Reel

This project is about constructing the server-side component of a web application. The application serves to provide users access to information about different movies, directors, etc.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all the movies /movies GET none a JSON object holding data about all the movies
Return a movie by its title /movies/:title GET none a JSON object holding data about the movie user searched for
Return a genre /movies/:genre/:genreName GET none a JSON object holding data about the genre user searched for
Return a director by name /movies/:directors/:directorName GET none a JSON object holding data about the director user searched for
Add new user to database /users POST a JSON object listing new user info a JSON object holding data about new user
Update user info /users/:id PUT a JSON object listing updated user info a JSON object holding data about updated user
Add movie to list of favorites /users/:id/:movieTitle POST none A message informing user the movie has been added to their favorites
Remove movie from list of favorites /users/:id/:movieTitle POST none A message informing user the movie has been removed their favorites
Delete user from directory /users/:id POST none A message informing user the profile has been deleted

JSON format for user info

  • {
  • "id": user's id number,
  • "name": "user's name",
  • "favoriteMovies": [array of favorite movies (can be left empty if desired)]
  • }

JSON format for movie info

  • {
  • "title": "movie title",
  • "director": { "name": "director name", "bio": "director bio"},
  • "genre": { "name": "genre name", "description": "genre description"}
  • }
