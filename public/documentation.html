For_Reel Documentation

This project is about constructing the server-side component of a web application. The application serves to provide users access to information about different movies, directors, etc.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all the movies /movies GET none a JSON object holding data about all the movies
Return a movie by its title /movies/:Title GET none a JSON object holding data about a single movie, containing its Title, Description, ImagePath, ID, Director(and relevant info), Genre(and relevant info), Featured Actors, and its "Featured" status. Example:
  • {
  • "Title": "Silence of the Lambs",
  • "Description": "A young FBI cadet must receive help of an incarcerated and manipulative cannibal killer.",
  • "Genre": {"Name":"Thriller", "Description": "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involves excitement and suspense in the audience."}
  • "Director" {"Name":"Jonathan Demme", "Bio": "Robert Jonathan Demme was an American director, producer, and screenwriter.", "Birth": "1944", "Death": "2017"}
  • "Actors": [an array of featured actors]
  • "ImagePath": "silenceofthelambs.png"
  • "_id": "651f1943b400a5ad7efb5a37"
  • "Featured": true
  • }
Return info about a genre /movies/:genre/:genreName GET none a JSON object holding data about the genre user searched for. Example:
  • {
  • "Name": "Fantasy",
  • "Description": "Fantasy is a genre of speculative fiction involving magical elements, typically set in a fictional universe and usually inspired by mythology or folklore.",
  • }
Return info about a director /movies/:directors/:directorName GET none a JSON object holding data about the director user searched for. Example:
  • {
  • "Name": "Ridley Scott",
  • "Bio": "Sir Ridley Scott is a British filmmaker. He is best known for directing films in the science fiction, crime, and historical drama genres.",
  • "Birth": "30-11-1937"
  • }
Add new user to database /users POST a JSON object listing new user info. Example:
  • {
  • "Username": "maryhadalittlelamb",
  • "Password": "lambsdontcry",
  • "Email": "lambs@gmail.com",
  • "Birthday": "01/01/1997"
  • }
 a JSON object showing data about new user. Example:
